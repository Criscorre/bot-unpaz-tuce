## Descripción

**Bot TUCE** es un asistente virtual multiplataforma diseñado para estudiantes de la **Tecnicatura Universitaria en Comercio Electrónico (TUCE)** de la UNPAZ.

Centraliza información académica, recursos y herramientas en una única solución, integrando inteligencia artificial y un sistema de perfiles profesionales.

---

## Funcionalidades

### Información Académica
- Horarios, aulas y docentes  
- Calendario académico  
- Plan de estudios  
- Correlativas  
- Sedes UNPAZ  

### Sistema de Talentos
- Perfiles profesionales de estudiantes  
- Búsqueda por habilidades  
- Sistema de valoración  
- Áreas: Desarrollo Web, Ads, Diseño, CM, PM  

### Herramientas
- Banco de recursos  
- OCR (texto desde imágenes)  
- Generación de documentos (PDF / Word)  
- Organización por materias  

### Asistente IA
- Consultas académicas inteligentes  
- Respuestas contextualizadas  
- Soporte en tiempo real  

### Novedades
- Actualizaciones automáticas mediante scraping

## Stack Tecnológico

**Backend**
- Python  
- pyTelegramBotAPI  
- OpenAI API  
- Firebase  
- BeautifulSoup  
- APScheduler  

**Integraciones**
- Node.js  
- Baileys  
- Axios  

**Infraestructura**
- Docker  
- Koyeb

## Seguridad

- Variables de entorno  
- Validación de entradas

## Objetivo

Optimizar el acceso a la información académica y facilitar la inserción profesional de los estudiantes mediante tecnología centralizada.
